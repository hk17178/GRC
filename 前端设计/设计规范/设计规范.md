## 0. 总则

- 本规范将 **5 套设计语言主题全部固化**为统一的设计令牌（Design Tokens）体系。**5 套全部保留、运行时自由切换**，并非五选一。
- **单一真源**：所有令牌定义于 [`tokens.css`](tokens.css)；可视化样式指南见 [`设计规范.html`](设计规范.html)（可切主题、实时显示令牌解析值）。
- **组件零耦合于主题**：组件只引用语义令牌（如 `var(--accent)`、`var(--surface)`），切主题仅替换 `<body>` 类名，组件代码零改动。

## 1. 令牌体系

令牌分两层：**基础令牌**（跨主题恒定）与**主题令牌**（按主题覆盖）。

### 1.1 基础令牌（恒定）

| 类别 | 令牌 | 值 |
|---|---|---|
| 状态色 | `--danger` / `--warning` / `--success` / `--info` / `--plum` | `#c0392b` / `#c2902f` / `#2f7a52` / `#2e6da4` / `#7c63c4` |
| 风险三色 | 高=`--danger` 中=`--warning` 低=`--safe` | 低风险用独立 `--safe`，不复用 `--accent`（见下） |
| 状态浅底 | `--*-tint` | 对应浅色底（标签/图标背景）|
| 字体 | `--font-sans` / `--font-serif` / `--font-mono` | Inter+系统中文 / Georgia+宋体 / 等宽 |
| 字号 | `--fs-display..--fs-2xs` | 24 / 19 / 15 / 14 / 13 / 12 / 11 / 10.5 px |
| 字重 | `--fw-regular..--fw-black` | 400 / 500 / 600 / 700 / 790 |
| 间距 | `--space-1..--space-8` | 4 / 8 / 12 / 16 / 20 / 24 / 32 / 40 px |
| 动效曲线 | `--ease-emphasized` | `cubic-bezier(.22,.61,.36,1)` |
| 动效时长 | `--dur-fast/base/slow/count/chart` | .2 / .3 / .6 / 1.1 / 1.2 s |

> 状态色刻意**不随主题变化**，保证“红=高风险 / 绿=通过”的认知在 5 套主题下完全一致。

### 1.2 主题令牌（按主题覆盖）

每个主题覆盖以下语义令牌：

| 令牌 | 含义 |
|---|---|
| `--bg` | 页面底色 |
| `--surface` / `--surface-border` | 卡片表面 / 描边 |
| `--blur` | 毛玻璃模糊（玻璃主题为 `blur(...)`，扁平主题为 `none`）|
| `--gloss` | 卡片顶部高光开关（0/1）|
| `--text-1..--text-4` | 主 / 次 / 辅助 / 最弱 文字 |
| `--border` / `--border-subtle` | 描边 / 更浅描边 |
| `--accent` / `--accent-strong` / `--accent-weak` / `--accent-tint` / `--accent-bright` | 主强调色族（品牌色）|
| `--safe` / `--safe-weak` | 低风险/安全色，**独立于 accent**（政务=政务蓝，其余=绿）|
| `--accent-gold` | 政务主题金色点缀（其余主题映射为各自强调色）|
| `--radius-sm/md/lg` | 圆角阶梯 |
| `--shadow-1/2` | 阴影层级 |
| `--font-display` | 标题/数字字体（宣纸=衬线，其余=无衬线）|
| `--aurora-op` + `--aurora-1/2/3` | 流光背景透明度与色彩 |

各主题具体取值见 [`tokens.css`](tokens.css)。

## 2. 组件规范（要点）

| 组件 | 关键令牌 | 说明 |
|---|---|---|
| 卡片 Card | `--surface` `--surface-border` `--shadow-1` `--blur` `--radius-lg` | 玻璃主题带顶部高光 `::before`（受 `--gloss`）|
| 按钮 Button | 主：`linear-gradient(--accent,--accent-strong)`；次：`--surface` + 描边 | 悬浮 `translateY(-2px)` |
| KPI 卡 | 图标底 `--accent-weak`；数值 `--font-display` 790 字重、等宽数字 | 入场数字滚动 |
| 风险标签 | 高=`--danger` 中=`--warning` 低=`--safe` | 始终“点 + 文字” |
| 状态 Status | 圆点 + 文字，颜色取状态色 | 进行中/逾期点可加脉冲 |
| 体系标记 Pill | `teal`=`--accent-weak`，`blue`=`--info`，`violet`=`--plum` | 体系/通道标识 |
| 表格 Table | 行 hover=`--accent-tint`，分隔=`--border-subtle`，编号=`--accent-strong` 等宽 | — |
| 输入/搜索 | `--surface` + 描边，聚焦描边转 `--accent-bright` | — |
| 导航项 | 选中=`linear-gradient(--accent,--accent-strong)` 白字 | hover 右移 3px |

## 3. 动效规范

| 场景 | 时长 | 曲线 | 实现 |
|---|---|---|---|
| 入场（页面/卡片）| `--dur-slow` .6s | emphasized | `fadeUp` 淡入上浮 + 子元素 stagger 0.04s |
| 卡片悬浮 | `--dur-base` .3s | emphasized | `translateY(-4px)` + `--shadow-2` |
| 数字滚动 | `--dur-count` 1.1s | easeOutCubic | requestAnimationFrame |
| 图表绘制（柱/环）| `--dur-chart` 1.2s | emphasized | 柱 `width`、环 `stroke-dasharray` 过渡 |
| 反馈（按钮/导航）| `--dur-fast` .2s | emphasized | transform / 颜色过渡 |

**原则**：动画只用 `transform` / `opacity`（走 GPU）；切换主题/页面时重放入场动画；全站受 `prefers-reduced-motion` 守卫，开启“减少动态”时自动降级为静态。

## 3.5 质感规范（Visual Quality）

让画面“看起来更贵”的硬性要求，编码时须遵守：

1. **色块用渐变不平涂**：KPI 图标底、进度条、图表色段用同色系线性渐变（亮→深），如 `linear-gradient(var(--accent-bright),var(--accent))`。
2. **描边 + 阴影叠加**：卡片 = 1px 半透描边 + 分层柔阴影（`--shadow-1` 为 2–3 层），不用单层硬阴影。
3. **材质分主题**：玻璃主题（晴空）= 半透表面 + `--blur` + 顶部高光（`--gloss`）；扁平/政务主题 = 实底 + 细描边 + 克制阴影；文字始终位于不透明蒙版之上。
4. **数字等宽**：数据一律 `font-variant-numeric:tabular-nums`；KPI 数值用 `--font-display` + 790 字重 + 负字距。
5. **图标 tint 块**：图标置于浅色块（`--accent-weak`/状态 tint）+ inset 顶部高光。
6. **悬浮反馈**：卡片 hover `translateY(-4px)` + 阴影升至 `--shadow-2`，玻璃主题可叠加流光 sheen。
7. **留白节奏**：间距 4 倍数阶梯，卡片内边距 18–22px。
8. **三忌**：忌纯黑文字、忌纯白生硬底、忌大面积纯色平涂。

---

## 4. 无障碍

- 状态色跨主题恒定，避免色彩语义漂移；
- 玻璃主题文字均位于不透明蒙版（卡片表面/scrim）之上，保证对比度；
- 已内置 `prefers-reduced-motion` 降级；
- 焦点态保留（输入聚焦描边高亮），后续深度页补全键盘可达性。

## 5. 落地与扩展

```html
<!-- 引入令牌 -->
<link rel="stylesheet" href="tokens.css">
<!-- 应用主题（5 套任选，可持久化到 localStorage） -->
<body class="t-teal">     <!-- 或 t-sand / t-glass / t-emerald / t-editorial -->
```

```js
// 运行时切换主题
document.body.className = 't-sand';
localStorage.setItem('grc-theme', 't-sand');
```

**新增主题**：复制任一 `body.t-*` 块，改令牌值即可，组件无需改动。
