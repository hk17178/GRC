# GRC 平台 · 用户操作手册（按角色）

**文档编号：** D5-1 ｜ **版本：** v1.0（草案） ｜ **日期：** 2026-06-23
**适用：** 平台各角色用户 ｜ **状态：** 待评审

> 本手册按角色组织常用操作，配合系统界面使用。所有"查看"默认仅限本子公司范围（集团管理层/只读审阅角色除外）。

---

## 一、通用
- **登录（多域）**：在登录页用 `DOMAIN\用户名` 或 `用户名@域` 输入账号，系统按域自动路由到对应身份源（各子公司/海外独立域）认证；用户唯一身份为（域+用户名）。也可使用单点登录。⚠️ 若你所在域临时不可用，仅本域登录受影响，已授予的权限以快照维持，集团聚合视图不中断。
- **导航**：左侧菜单按"概览 / 业务 / 系统管理"分组；点击进入对应模块。业务菜单顺序为：外部审计 → 内部审计 → 风险评估 → 法规跟踪 → 监管事项 → 合规清单 → 制度体系 → AI 智能问答 → 第三方供应商（外部审计、内部审计为两个独立菜单）。
- **语言切换**：顶栏右上角"语言"可在中文/英文间切换；日期、数字、时区按本地化显示，部分内容支持双语呈现。
- **主题**：右上角"主题"可切换界面风格（不影响数据）。
- **AI 助手**：左下角悬浮按钮，随时提问（回答按你的权限范围）；也是"建议与反馈"的提交入口之一。
- **通知**：右上角铃铛查看；企微/飞书/邮件按规则接收。

## 二、普通员工（制度签署方）
- **阅读制度**：制度体系 → 查看适用于你的制度。
- **签署确认**：收到提醒后，打开制度 → 阅读 → 点"已阅读并知悉确认"。⚠️ 逾期未确认会提醒你及上级。

## 三、系统/业务责任人
- **填写风险自评（A-T-V 录入）**：风险评估 → 我的评估 → 填写表单。按"资产(Asset)/威胁(Threat)/脆弱性(Vulnerability)"三要素录入，系统据此计算风险等级。风险等级采用五级：**极低 / 低 / 中 / 高 / 极高**。可上传佐证 → 提交（进入审核）。
- **风险处置与残余风险**：对评出的风险选择处置方式（降低/转移/规避/接受）→ 录入处置措施与计划 → 处置后填写**残余风险**等级（仍为极低/低/中/高/极高）。
- **管理层接受签字**：当残余风险选择"接受"或仍处于高/极高时，需走"管理层接受"流程，由相应层级管理者电子签字确认后归档。
- **承接整改**：整改任务 → 处理 → 完成后上传证明 → 等发起人验证。⚠️ 未上传证明不可关闭。

## 四、部门级评估审核人
- **审核评估**：风险评估 → 待审核 → 查看 → 通过/退回。
- **确认整改验证**：整改任务 → 待验证 → 核对证明 → 验证通过/退回。

## 五、信息安全负责人（评估侧）
- **审批报告**：风险评估 → 报告审批 → 复核 → 签批生效。
- **配置打分模型/风险矩阵**：由模板配置入口设置（见管理员手册）。

## 六、审计执行人员（审计管理：内/外审/监管检查）
> 模块由"内部审计"升级为"**审计管理**"，按 audit_type 区分：内审 / 外审 / 监管检查 / 认证审计。
- **执行检查表**：审计管理 → 我的审计 → 填写检查表 → 记录发现（描述/依据条款/严重度/上传证据）。
- **整改跟踪**：发现项转整改（复用整改任务）。
- **登记外审任务**：审计管理 → 外审 → 新建外审任务，选择审计类型（外部审计/监管检查/认证审计/等保测评），录入外部机构、范围、时间窗。
- **录入外部发现**：在外审任务下录入外方提出的发现/问题项（描述/依据/严重度/整改要求/期限/证据），可转整改任务跟踪闭环。
- **对外回函三段漏斗**：对需对外回复的发现，按三段推进状态——①已提交外部机构 → ②外方受理 → ③外方确认关闭；每段记录时间与回函附件，看板可见各发现停留阶段。
- **认证有效期跟踪**：认证审计任务记录证书与有效期，到期前自动预警，支持续证/复审排期。
- **等保测评全过程视图**：等保测评以全过程视图呈现（定级备案 → 测评实施 → 整改 → 复测 → 结论），逐环节留痕与证据归集。

## 七、审计计划制定人 / 报告审批人
- **制定计划**：审计管理 → 审计计划 → 新建 → 排期 → 关联对象与检查表（可指定内审/外审/监管检查/认证审计类型）。
- **签批报告**：审计管理 → 报告 → 复核签批。

## 八、合规分析人员 / 法规库维护人员
- **处理法规更新**：法规跟踪 → 更新流 → 查看变更摘要 → 判断对内部制度影响。
- **维护映射**：登记法规条款 ↔ 制度条款映射。

## 九、供应商管理人员
- **维护供应商**：第三方供应商 → 台账 → 新增/编辑；发起准入/续约/退出评估。
- **事件触发复评**：登记外部事件（如负面舆情）→ 发起复评。
- **SLA 跟踪**：查看 SLA 状态与到期提醒。

## 十、监管事务专员（监管事项）
> 监管事项为独立一级业务模块。注意：IR/SOC 主体不在本平台，本平台仅承载"重大事件监管报送"。
- **报送日历**：监管事项 → 报送日历 → 查看法定时限报送任务，临近时限自动预警，按完成情况打卡。
- **监管问询**：监管事项 → 问询 → 登记监管机构问询 → 分派承办 → 录入答复与归档。
- **处罚/约谈台账**：监管事项 → 处罚约谈台账 → 登记处罚或约谈事项、整改要求与闭环状态。
- **重大事件报送**：监管事项 → 重大事件报送 → 按口径填写并报送重大事件（仅报送动作在本平台）。
- **年度合规计划**：监管事项 → 年度合规计划 → 编制/分解/跟踪年度合规工作项。

## 十一、合规管理人员（合规清单 / ROPA / 取证）
- **合规清单**（原合规义务清单，独立一级菜单）：合规清单 → 查看/维护由法规分解出的合规义务及责任人、到期、满足状态。
- **个人信息处理记录 ROPA**：合规 → ROPA → 登记/维护处理活动记录（处理目的、类别、流向、留存等）。
- **合规卷宗导出**：合规 → 卷宗 → 选择范围导出合规卷宗（含证据清单）。
- **按法规条款反向取证**：在某法规条款上选择"反向取证"，系统列出关联的制度条款、控制、审计发现与证据，支持一键导出。

## 十二、子公司/集团管理层
- **查看看板**：合规态势 → 按子公司/时间下钻查看风险、整改、审计态势。⚠️ 子公司管理层仅本子公司；集团管理层可看全集团。
- **跨子公司对比下钻**：合规态势 → 选择多家子公司并排对比 → 逐层下钻到具体评估/发现/整改。
- **合规风险指数下钻**：从合规风险指数总览逐级下钻到子公司/业务域/单项风险（等级仍为极低/低/中/高/极高）。
- **全文检索**：顶栏全文检索框，跨制度/法规/审计/监管事项检索，结果按权限过滤。

## 十三、AI 智能问答（全角色）
- 提问示例："对比 A、B 子公司近两次评估的高/极高风险项变化"。
- 回答会标注数据来源；超出你权限范围的数据不会返回。

## 十四、用户访问复核（UAR）经办/复核人
- **发起复核**：UAR → 新建复核活动 → 选择范围（系统/子公司/特权账号）→ 生成待复核清单。
- **逐项复核**：对每个账号权限选择"保留 / 回收"，填写理由。
- **签字确认**：复核人对复核结论电子签字归档；回收项生成整改/工单跟踪执行。
- **越权与特权 KRI 下钻**：UAR 看板查看越权访问、休眠特权账号等 KRI 指标，点击可下钻到具体账号与明细。

## 十五、审计与合规审阅（平台操作审计日志）
- **操作审计日志查询**：系统管理 → 操作审计 → 按用户/时间/模块/动作检索平台操作留痕（含配置变更、AI 配置变更、对外推送等出站动作），支持导出。

---

## 十六、建议与反馈（全角色）
- **双入口提交**：①系统管理 → 建议与反馈 → 新建；②AI 助手对话中直接提交 BUG/功能建议。系统会自动格式化你的描述。
- **查看分类与可行性**：提交后可在"我的反馈"查看系统分类（BUG / 建议）、AI 可行性初评结论（以人工复核为准）及处理进展。
- **进展跟踪**：被采纳的条目会进入周期分析并推送管理员；涉及对外同步的，经脱敏与管理员审批后推送至代码仓库"反馈与建议"目录（出站审计可查）。

## 十七、常见问题
| 问题 | 说明 |
|---|---|
| 看不到某子公司数据 | 受组织权限限制，属正常 |
| 表单提交不了 | 检查必填项；网络/会话；草稿会自动保存 |
| 没收到通知 | 确认通知偏好与通道；联系管理员核对规则 |
